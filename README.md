# Energy Bill Extractor

A mobile-friendly browser app for reading monthly kWh values from an electric bill graph, calculating annual usage and cost, and exporting a summary image.

## Deploy

This project is configured for Netlify.

- Build command: leave blank
- Publish directory: `outputs/energy-bill-extractor`

The app runs entirely in the browser. Uploaded bill images and PDFs are processed locally on the user's device.
